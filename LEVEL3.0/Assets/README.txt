UNITY PHYSICS
3 вида:

1. Dynamic  / Динамични
2. Kinematic/ Кинематичен
3. Static   / Статични

1. Динамичните притежават свойства за: гравитация, колизия + взаимодействие с други динамични обекти
2. Кинематичните обекти са като статичните, но те могат да се движат (могат да си взаимодействат с динамичните/ използват се за платформи)
3. Статичните обекти НЕ се движат / Динамичните обекти могат да се блъскат (колизия) с тях
